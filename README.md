- 2- Comment modifier votre travail ?

      Vous allez voir une répétition du même code, vous avez juste à modifier le

    ```Html
      <div id="news">
        <h2>
          News !!
        </h2>
        <p>
          Mon article qui est trop bien !
        </p>
        <h3>Source : <a href="#">blabla</a>
        </h3>
      </div>
    </div>
    ```
